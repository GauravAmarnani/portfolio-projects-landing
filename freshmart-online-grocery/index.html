FreshMart – Online Grocery System

Live Demo Source Code

FreshMart is an online grocery platform designed to provide users with a smooth purchasing experience while offering products at competitive prices without compromising quality.

The system enables users to browse products, place orders, and manage interactions through a scalable, service-oriented backend built using modern microservices principles.

Engineering Focus

The project focuses on building a modular and extensible backend capable of supporting real-world e-commerce workflows while maintaining clean separation of responsibilities across services.

Architecture & Design

  • REST-based microservices architecture
  • Service-to-service communication using Feign Clients
  • Service discovery and centralized configuration
  • Clear domain separation across Admin, Customer, and Shop services

Key Capabilities

  • User-friendly product browsing and order placement
  • Backend services designed for scalability and maintainability
  • Clean API contracts for frontend integration
  • Support for future feature expansion without architectural rework

Technology Stack

  • Backend: Java, Spring Boot, REST, Microservices
  • Frontend: Angular
  • Communication: Feign Client
