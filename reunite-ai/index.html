ReUnite AI - Finding Missing People

Live Demo Source Code

ReUnite AI is a machine-learning-driven system designed to assist in reconnecting missing individuals with their families by leveraging facial recognition and age progression technologies.

The project addresses a real societal challenge by combining data science, backend engineering, and applied machine learning in a production-style system.

Problem Focus

Identifying missing persons becomes increasingly difficult as time passes. ReUnite AI models how facial features evolve over time, enabling the system to match historical images with more recent representations.

Core Technical Approach

  • Facial recognition using Principal Component Analysis (PCA)
  • Age progression using Generative Adversarial Networks (GANs)
  • Model training on diverse facial image datasets
  • Evaluation across varying age ranges and time gaps

Backend & System Design

  • Backend services built using Python and Django
  • Secure database connectivity and encrypted storage of sensitive data
  • Clean separation between ML inference and application logic
  • Designed for extensibility as models and datasets evolve

Technology Stack

  • Backend: Python, Django
  • Machine Learning: PCA, GANs, Keras, TensorFlow
  • Database: MySQL
